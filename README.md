# exam-analysis (試題分析系統)

本專案旨在提供一個美觀、直觀且現代化的網頁平台，用以匯入、分析及視覺化考試成績與試題指標。

## 專案目標

提供教師或教育研究者一個功能強大且易於使用的試題分析工具，包含：
- **試題難難度與區辨度分析**：計算難度指數 ($P$) 與區辨度指數 ($D$)，評估試題品質。
- **成績分佈與統計**：視覺化呈現班級成績分佈（直方圖、盒鬚圖等）、平均數、中位數、標準差與信度指標。
- **互動式報表**：提供動態、可互動的圖表與數據篩選功能。
- **響應式與高質感設計**：完美支援各式螢幕尺寸，並具備流暢的微動畫與高品質的視覺美學。

## 技術底座

- **核心技術**：HTML5, CSS3 (Vanilla CSS), JavaScript (ES6+)
- **預計整合套件**：
  - 圖表繪製：Chart.js / ApexCharts (後續決定)
  - 圖示庫：Lucide Icons / FontAwesome

## 快速開始

### 本地開發與執行

1. 克隆此儲存庫：
   ```bash
   git clone <repository-url>
   cd exam-analysis
   ```
2. 使用瀏覽器開啟 `index.html`，或使用 VS Code Live Server 啟動本地開發伺服器。

## 專案規範與安全

本專案將嚴格遵守 `ANTIGRAVITY.md` 所規範之流程：
- 嚴禁上傳真實學生個資與真實姓名。
- 專案開發應以視覺美學與現代設計感為優先考量。
